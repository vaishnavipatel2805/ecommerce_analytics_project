# 🛒 E-commerce Analytics & Customer Intelligence System

## 📌 Project Overview

This project focuses on analyzing a real-world e-commerce dataset to uncover insights related to revenue trends, customer behavior, delivery performance, and customer satisfaction. The goal is to simulate a business scenario and provide actionable recommendations to improve decision-making and overall customer experience.

---

## 🎯 Problem Statement

To analyze an e-commerce platform’s multi-dimensional data to identify key factors affecting revenue, customer behavior, delivery performance, and customer satisfaction, and to derive actionable insights that can improve business performance and user experience.

---

## 📊 Dataset

The dataset used for this project is:

* Brazilian E-Commerce Public Dataset by Olist

It contains information on:

* Orders
* Customers
* Payments
* Reviews
* Order items

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (Data querying and analysis)
* **Power BI** (Dashboard and visualization)

---

## 🔄 Project Workflow

1. **Data Understanding**

   * Explored multiple tables and their relationships

2. **Data Integration**

   * Merged multiple tables to create a master dataset
   
3. **Data Cleaning & Preprocessing**

   * Handled missing values
   * Converted data types
   * Prepared datasets for analysis

4. **Exploratory Data Analysis (EDA)**

   * Analyzed trends in sales, customers, and delivery

5. **Feature Engineering**

   * Created new features such as total price and delivery time

6. **Dashboard Creation**

   * Built interactive dashboards using Power BI
